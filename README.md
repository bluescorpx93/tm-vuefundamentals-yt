### Vue Tryout

To Test This

- Install Package Dependencies
```sh
$ cd tm-vue-basics-yt
$ yarn install 

# or with npm
$ npm install
```

- Run in Prod Mode
```sh
$ cd tm-vue-basics-yt
$ yarn start-app
```

- Run in Vue Dev Mode
```sh
$ cd tm-vue-basics-yt
$ yarn start
```

*Don't forget to run **yarn build** after updating the vue source*
